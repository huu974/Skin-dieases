# 皮肤病智能诊断系统 - 前端界面

## 文件结构

```
frontend/
├── main_app.py              # 主入口页面（仪表盘）
└── pages/
    ├── 1_诊断分析.py        # 图片上传与分析
    ├── 2_智能对话.py        # 对话聊天界面
    ├── 3_历史记录.py        # 诊断历史记录
    ├── 4_预防建议.py        # 预防建议
    └── 5_用户中心.py        # 用户信息管理
```

## 功能特性

- ✅ 图片上传与分析 + 实时检测 + 多图对比
- ✅ 智能对话聊天界面
- ✅ 诊断历史记录管理
- ✅ 皮肤病预防建议
- ✅ 用户信息管理
- ✅ 多语言支持（中/英）
- ✅ 诊断报告导出
- ✅ 进度指示器
- ✅ 黑暗模式切换

## 运行方式

在项目根目录下运行：

```bash
streamlit run frontend/main_app.py
```

## 注意事项

- 所有页面共享 `st.session_state` 中的用户信息和主题设置
- 原有的 `app.py` 保持不变，作为备用
